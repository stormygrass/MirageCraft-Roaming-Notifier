# MirageCraft Notifier
**A simple client-side mod that listens for roaming events and alerts the user. If you wish to change the volume of the alert use the "voices/speech" slider in minecraft's music settings.**

A config menu will be added in the future and you will be able to customise the notification alerts and target them for certain legendary pokemon etc.


## FAQs
**Q: Why should I use this mod over options like ChatNotify etc.?**
*A: This mod offers ease of use and is readily configured for the MirageCraft server. This means that it can be more lightweight. It also plays a custom sound for the notification so theres no chance you will get the notification mixed up with a vanilla minecraft sound. Plus it only triggers from the server so it wont get tricked by discord webhooks or players who might troll with keywords.*


## Building
use ``.\gradlew build`` if you wish to build it yourself.
